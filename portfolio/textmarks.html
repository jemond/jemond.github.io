Textmarks Drupal Module

The Textmarks Drupal module allows Drupal users leverage the Textmarks API without any programming required.

Textmarks is a free SMS gateway service that allows users to create a textmark that mobile users can subscribe too, and through which the content provider can send messages to subscribers.

This module was built to power the mobile integration of the Navigate Hollywood site.

Drupal.org project page.

Core features:

  • A settings screen to track your API connection settings
  • A screen to send messages to all subscribers of your textmark
  • A screen to subscribe a mobile number to your textmark (the mobile user must opt-in)

My role:

  • Developer
  • Project maintainer

Core technologies:

  • PHP 5
  • Drupal API
  • Textmarks API
