Better Support Forums

Better Support Forums is an easy to use support forum software engine for a website. Released as a blog post, the software is an attempt to take approach of FogCreek Software's discussion forums in a new direction.

Core features:
  • Tagging of posts for folksonomy;
  • Attach screen shots to posts;
  • Automatic related ticket linking;
  • Full text search;
  • Clean URLs for ticket and category links; and,
  • Customizable look and feel using layout files.
My role:
  • Lead developer
  • Interface designer
  • Application architect
  • Database architect
Core technologies:
  • PHP 5.0
  • MySQL 5.0
