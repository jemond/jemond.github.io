Milo Web Chat

Milo Web Chat (formerly University Web Chat) is a plug-in-less live chat tool for admissions recruiting. Designed for non-technical users, the easy to use chat room setup wizard automatically detects client logos and colors, and brands the chat room accordingly allowing instant setup.

Having left beta in March 2008, Milo Web Chat is in production use and is profitable.

Core features:

  • Setup and customize chat rooms with an easy to use wizard
  • Room setup script searches and attaches client logos automatically
  • Clean URLs for simplified publication in marketing materials
  • Cross-browser, cross-platform compatible
