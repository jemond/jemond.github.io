Student Affairs Alumni Tracking System

The USC School of Public Policy, Planning, and Development (USC SPPD) needed a tool to help connect their current students seeking mentorship and internships with SPPD's vast alumni network of working professionals.

We created a web-based application that integrates with USC's alumni database (Oracle's PeopleSoft) and provides additional reporting, tracking, and filtering on top of that data. The system is called the Student Affairs Alumni Tracking System (STATS).

Successfully launched in March, 2009.

My role:

  • Project & client manager
  • Technical architect
  • Interface designer

Core features:

  • Integration with Oracle PeopleSoft
  • Full-featured search and filter controls
  • Reports for alumni overview, mentorship availability, and alumni expertise
  • Contact log, event tracking, and email tool

Core technologies:

  • ASP.NET
  • Microsoft SQL Server
